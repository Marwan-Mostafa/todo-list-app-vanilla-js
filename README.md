# 📝 To-Do List App

A simple and interactive To-Do List application built using Vanilla JavaScript.

## 🚀 Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Prevent empty inputs
- Dynamic DOM manipulation

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (Vanilla JS)

## 🎯 Purpose
This project is part of my journey to master JavaScript by building real-world applications.
